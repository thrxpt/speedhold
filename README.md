# <img src="assets/icon.png" width="32" height="32" align="center" /> SpeedHold

Hold a hotkey to speed up any video. Release to return to normal speed.

## How to Use

1. Open any video.
2. Hold **S** (default) to speed up.
3. Release to return to 1x speed.

_Configure the hotkey and speed multiplier in the extension popup._

## Features

- **Hold to Speed Up**: Instant playback boost on any video.
- **Visual Overlay**: A clean indicator shows current speed.
- **Universal**: Works on YouTube, Netflix, Twitch, and any other site using HTML5 video.
